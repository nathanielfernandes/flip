## flip

cli tool to quickly convert an image to a gif, with some other goodies.

that's all it does.

**usage**

```zsh
flip some-image.png
>> some-image.gif: flipping...
>> some-image.gif: done in 145.72ms
```

**installation**

```zsh
cargo install --git https://github.com/nathanielfernandes/flip
```
